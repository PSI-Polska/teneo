List of test XML Schemas

The integration layer is tested with a number of different test cases. Some of these test cases are based on EMF classes generated from XML Schemas and ecore files. A subset of the used XML Schemas is provided below so they can be used to test the integration layer. For more details please get the test project from subversion (see here).

  • Library.xsd: the library example
  • Catalog.xsd: a simple product catalog
  • PrimerPO.xsd: the primer example used by XML Schema and others
  • Forum.xsd: a simple forum
  • sun_books.xsd: example as provided by Sun for Jaxb (see here)
  • play.xsd: example on how to model theatre plays, (copied from here), the test case uses the following data file: play.xml
