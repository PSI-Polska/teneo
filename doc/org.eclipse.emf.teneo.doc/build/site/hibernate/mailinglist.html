EMFT Newsgroup

Elver persistency is moving to Eclipse.org as a subproject in EMFT. Support for Elver/Teneo is given on the EMFT newsgroup: news://news.eclipse.org/eclipse.technology.emft.

Mailing List

Support is provided through a mailinglist. You can subscribe to this mailinglist by sending an email to: Subscribe. Unsubscribing is done through the following link: Unsubscribe. The traffic on the mailinglist is probably pretty low.

The mailing list is used for the EMF - JDO as well as the EMF - Hibernate integration.

The mail archive can be viewed here (web) or here (news, nntp).

How we can help you

We will do our best to answer your questions as promptly as possible. We can help you in the best way if you include the following information:

  • The genmodel file
  • The package.jdo
  • The stack trace
  • The log file (see here on where to find it)
  • The version of Elver Store, Eclipse, EMF, jdk, and database
